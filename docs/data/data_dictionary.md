# Diccionario de datos

## Base de datos 1

| Variable          | Tipo         | Contenido principal                            | Función                                           |
|--------------------|--------------|------------------------------------------------|--------------------------------------------------|
| data_root          | Carpeta      | Carpeta raíz del dataset                       | Punto base para construir rutas                  |
| gz2_fn_path        | Archivo CSV  | IDs ↔ nombres de archivos                      | Relaciona imágenes con galaxias                  |
| images_path        | Carpeta      | Archivos .jpg o .png                           | Contiene las imágenes del dataset                |
| gz2_hart16_path    | Archivo CSV  | Etiquetas (morfología, espiral, barra, etc.)   | Usado como variable objetivo para clasificación
